AE 김○○

AE 정○○
시트 열면 머리가 아파요. 1만 개 행 필터링하는 데만 시간이 걸리는데, 저는 그냥 제 매장 성과만 보고 싶거든요.
👨
AE 우○○
저성과 매장이 한두 곳이 아니라서 부스팅 요청을 일괄로 하고 싶어요. 슬랙 들어가서 워크플로 요청할 시간도 없거든요.
👩
AE 조○○
지역마다 성과가 다르다는데 체감이 잘 안 돼요. 이것도 스프레드시트로 봐야 해서 어디가 약한지 딱 보이질 않더라고요.
🔍

기존 방식 vs. Catch Ad View

운영 방식의 변화

기존 방식에서 각 항목이 어떻게 바뀌는지 정리했습니다.

업무 항목 기존 방식 Catch Ad View
광고 성과 확인 1만 개 이상의 행이 쌓인 시트에서 광고상품명 열을 직접 필터링 → 매장별 ROAS 수동 탐색 ✓ 로그인 즉시 ROAS·연속참여율·매장 수만 압축 표시
저조 매장 파악 조건부 서식 확인 또는 수동 필터링 ✓ ROAS 300% 미만 자동 분류 및 별도 탭 제공
지역별 성과 분석 시트에서 직접 불가 → 광고비즈니스셀에 요청 후 결과 대기 ✓ AE가 직접 지도 클릭으로 시·도 → 시·군·구 → 상권 드릴다운
부스팅(리스팅·지도·패키지) 요청 매장마다 시트 접속 또는 워크플로 개별 실행 ✓ 체크박스로 복수 매장 선택 → 원클릭 일괄 요청
결제/계약 미완료 관리 개별 확인 또는 슬랙 알림 직접 확인 ✓ D-Day 기준 통합 현황 페이지 (마스터 계정 담당자별 필터 포함)

Expected Value

업무 구조의 변화

현재 AE 1인 기준 계약 확인·성과 파악에 주 5.6시간, 매장 운영성 업무에 주 8시간이 투입되고 있습니다. Catch Ad View는 이 두 영역의 처리 방식을 구조적으로 바꿉니다.

AE 주간 업무 시간 분포 (현재)
계약 확인 및 광고 성과 파악 5.6h

매장 운영성 업무 8.0h
